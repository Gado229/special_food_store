## Execution Procedure

>> $ git clone https://github.com/Gado229/special_food_store.git <br>
>> $ cd special_food_store <br>
>> $ bundle install <br>
>> $ rails db:create <br>
>> $ rails db:migrate <br>
>> $ rails s <br>
>> $ rails s -b 0.0.0.0 (Vagrant users) <br>
